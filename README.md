# concurrentQueue
Implementation of a thread safe queue based on linked list with basic and low-level synchronization tools like Semaphores and Atomic data types in java
for handling multiple consumers and producer threads.


### Features
* priority based queue
* simple TTL mechanism
* limit the total size of messages in queue
* write and read the queue to disc
* simultaneous access to both producer and consumer
* both bounded and unbounded capacity
* non blocking put and take methods
